# Skills — Catálogo

Receitas executáveis para tarefas repetitivas. Substituem o standard na conta do context budget.

## Skills aprovadas (`/approved/`)

| Skill | O que faz |
|---|---|
| `new-controller.md` | Passo a passo para criar novo controller REST com CORS, validação e envelope |
| `new-service.md` | Passo a passo para criar service com Query Builder + espelho AR |
| `new-migration.md` | Passo a passo para criar e aplicar migration SQLite-safe |

> As skills acima ainda não foram criadas — adicionar conforme necessidade.

## Como propor uma nova skill

1. Identificar uma tarefa repetida ≥ 3 vezes
2. Criar arquivo em `skills/approved/<nome>.md` com:
   - Objetivo
   - Pré-requisitos
   - Passo a passo executável
   - Checklist de validação
3. Registrar neste README
