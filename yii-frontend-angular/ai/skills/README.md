# Skills — Catálogo

Receitas executáveis para tarefas repetitivas. Substituem o standard na conta do context budget.

## Skills aprovadas (`/approved/`)

| Skill | O que faz |
|---|---|
| `new-component.md` | Passo a passo para criar novo componente AngularJS |
| `new-service.md` | Passo a passo para criar novo service com unwrap de envelope |
| `new-state.md` | Passo a passo para adicionar novo estado UI-Router |

> As skills acima ainda não foram criadas — adicionar conforme necessidade.

## Como propor uma nova skill

1. Copiar `templates/skill.template.md`
2. Preencher com o passo a passo executável
3. Salvar em `skills/approved/` após validação
